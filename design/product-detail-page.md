# Intro
This document should be used as a general design reference for the product detail page for the Cubby 2 Bed. This page should be thought of a subpage of a larger site used for researching Cubby and it's products.

## Colors
* background: #E6E7EF
* containers: #FFFFFF
* font: #374663
* highlight 1: #FACD7A
* highlight 2: #60BFE8

## Requirements
1. All design should in some way address concerns found in the [pitch](../pitch.md) document while primarily focusing on implementing the recommendation.
2. This page should showcase the Cubby 2 Bed. Details for this product can be found in the [product](./product.md) document.
3. This page should include images of the product and related products. Images can be found in the [images](./images/) directory.
4. This page should include links to important documents. PDFs can be found in the [documents](./documents/) directory.
5. This page should primarily focus on helping users locate data quickly. All details from the section below must be included in the page.
6. This page should look modern but professional. Cubby is a modern, exciting alternative in the medical device space, the site should reflect that.
7. The primary metric for success is engagement with Suppliers. We need to encourage sale #1 and build a path to sale #5.
